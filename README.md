# من هنا نعلم — This Is Where We Learn

Interactive trilingual web app based on **"Min Huna Na'lam"** (This Is Where We Learn) by **Sheikh Mohammed al-Ghazali** (1917-1996).

## About the Book

"Min Huna Na'lam" is Sheikh al-Ghazali's scholarly response to Khalid Muhammad Khalid's "From Here We Begin" which called for separating religion from the state. Al-Ghazali proves that Islam is both religion and state, refutes secularism claims, and corrects misconceptions about Islamic governance. Written with scholarly rigor yet respectful disagreement.

## Features

- **Trilingual**: Arabic, English, French
- **20 Knowledge Cards**: Islam as a complete system — governance, economics, society, knowledge, ethics, worship, and more
- **Discovery Path**: Progressive learning journey with XP and badges
- **Quiz**: "Who Wants to Be a Scholar?" format with 3 difficulty levels and lifelines (50/50, Ask the Sheikh, Istikhara)
- **XP System**: +10 per card read, +5 per correct answer, +20 per quiz completion
- **5 Badges**: Beginner, Reader, Scholar, Persistent, Expert
- **Narrator**: Audiobook mode with karaoke, speed/pitch control, sleep timer, duo reading (AR+FR)
- **3 Themes**: Teal (light), Night (dark), Ocean (green)
- **PWA**: Installable progressive web app
